public PropertyDescriptor[] 
                           getPropertyDescriptors()

"Gets the beans PropertyDescriptors.
Returns:
An array of PropertyDescriptors describing the 
editable properties supported by this bean. May 
return null if the information should be obtained
by automatic analysis. 

If a property is indexed, then its entry in the 
result array will belong to the 
IndexedPropertyDescriptor subclass of 
PropertyDescriptor. A client of 
getPropertyDescriptors can use "instanceof" to 
check if a given PropertyDescriptor is an 
IndexedPropertyDescriptor."
